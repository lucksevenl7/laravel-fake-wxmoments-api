使用步骤

- step 1

	将
    ```
	"repositories": [
	        {
	            "type": "vcs",
	            "url": "https://github.com/lucksevenl7/laravel-fake-wxmoments-api.git"
	        }
	]
	```
	添加到项目的composer.json文件中
- step 2
 composer require jdsf/miniforum dev-master
- step 3
 php artisan jdsf:miniforum:install
